ノートパソコン・準備物

PC・タブレットについて

オンライン授業やレポート作成のため、ノートPCが必要です。医学科では講義資料が電子配布されるため、iPadなどのタブレットも推奨されています。

推奨スペック（目安）

  • CPU: Core i5 / Ryzen 5以上
  • メモリ: 8GB以上（16GB推奨）
  • SSD: 256GB以上
  • OS: Windows / Mac どちらでもOK

購入時期と場所

購入時期：入学式後のITオリエンテーション前まで

購入場所：メーカー直販（学割あり）、大学生協、家電量販店など
